# Compass Forge

> AI development platform

## Overview

Compass Forge is the AI development platform powering Compass Brand's intelligent tools and services.

## Components

| Component                            | Description                                    |
| ------------------------------------ | ---------------------------------------------- |
| [forge-rag](./forge-rag)             | Custom RAG system (Ant Colony V2 architecture) |
| [forge-providers](./forge-providers) | Multi-provider LLM abstraction layer           |
| [compass-engine](./compass-engine)         | Central development tools (BMAD, Claude Code config, workflow scripts)                        |

## Quick Start

```bash
git clone --recurse-submodules https://github.com/Compass-Brand/compass-forge.git
cd compass-forge
```
